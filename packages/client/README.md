# resona-client

HTTP client library for the [Resona](https://github.com/ecsplico/resona) API. Wraps `resona-api` job/profile endpoints behind a small `ResonaClient` class, reads `RESONA_API_URL` / `RESONA_API_KEY`, and includes config plumbing for the `resona` CLI (engine entries, SSH tunnels, defaults).

See the [main repository](https://github.com/ecsplico/resona) for documentation.
