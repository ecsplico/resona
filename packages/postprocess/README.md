# resona-postprocess

Postprocessing pipeline for [Resona](https://github.com/ecsplico/resona) — composable steps for regex replacements, LLM transforms (via litellm), and structured extraction. Configured via JSON `Profile` files; ships a bundled default profile tuned for German medical dictation.

See the [main repository](https://github.com/ecsplico/resona) for documentation.
