# resona-engine-voxtral

HuggingFace Transformers backend for [Resona](https://github.com/ecsplico/resona) — supports Voxtral, Whisper, and other ASR models from the HF Hub via the `pipeline("automatic-speech-recognition", ...)` interface.

See the [main repository](https://github.com/ecsplico/resona) for documentation.
