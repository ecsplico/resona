# resona-engine-faster-whisper

CTranslate2 (faster-whisper) backend for [Resona](https://github.com/ecsplico/resona). The default and recommended engine — lower memory footprint and faster inference than vanilla OpenAI Whisper, with similar quality.

Install with `resona-engine-server` to expose it over HTTP/WS, or call it directly via `resona_asr_core.registry`.

See the [main repository](https://github.com/ecsplico/resona) for documentation.
