# resona-cloud-tts

Cloud text-to-speech providers for [Resona](https://github.com/ecsplico/resona). Uniform wrappers over OpenAI, ElevenLabs, and Deepgram text-to-speech APIs, returning a `SpeechResult` with audio bytes and content type.

See the [main repository](https://github.com/ecsplico/resona) for documentation.
