# resona-cloud-stt

Cloud speech-to-text providers for [Resona](https://github.com/ecsplico/resona). Thin, uniform wrappers over Deepgram, ElevenLabs, and OpenAI Whisper APIs — each provider returns the same `TranscriptionResult` TypedDict so the rest of the stack treats them like local engines.

See the [main repository](https://github.com/ecsplico/resona) for documentation.
