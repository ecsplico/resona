# resona-asr-core

Lean ASR contracts for [Resona](https://github.com/ecsplico/resona): the `Transcriber` protocol, entry-point–based engine registry, audio loader, and VAD-based live transcriber. No FastAPI, no I/O beyond audio decoding — pure interface and helpers, suitable for embedding in non-server contexts.

See the [main repository](https://github.com/ecsplico/resona) for documentation.
