Core API

replacements.py — same pure function, new home
def apply_replacements(
    text: str,
    replacements: list[dict[str, str]],
) -> str:
    """Apply regex replacements in order. Same logic as before."""
    for r in replacements:
        text = re.sub(r["name"], r["replacement"], text, flags=re.IGNORECASE)
    return text
pipeline.py — composable postprocessing
from typing import Callable

# A step is just a function: str → str
PostprocessStep = Callable[[str], str]

class PostprocessPipeline:
    """Chain of text transformations applied after transcription."""

    def __init__(self):
        self._steps: list[tuple[str, PostprocessStep]] = []

    def add(self, name: str, step: PostprocessStep) -> "PostprocessPipeline":
        self._steps.append((name, step))
        return self

    def run(self, text: str) -> str:
        for name, step in self._steps:
            text = step(text)
        return text
sources.py — load replacements from DB or file
from pathlib import Path
import json

def load_replacements_from_file(
    path: Path | None = None,
) -> list[dict[str, str]]:
    """Load from ~/.resona/replacements.json (or custom path)."""
    path = path or Path.home() / ".resona" / "replacements.json"
    if not path.exists():
        return []
    return json.loads(path.read_text())

def build_pipeline(
    replacements: list[dict[str, str]],
) -> PostprocessPipeline:
    """Convenience: create a pipeline with replacements step."""
    pipeline = PostprocessPipeline()
    if replacements:
        pipeline.add(
            "replacements",
            lambda text: apply_replacements(text, replacements),
        )
    return pipeline

How Each Consumer Uses It

resona-api (server mode)

# tasks_transcribe.py
replacements = get_active_replacements(db)
pipeline = build_pipeline(replacements)

result = engine_client.transcribe(filepath)
md = pipeline.run(result["text"])

Replacements from DB, postprocessing after engine returns raw text.

resona-cli (local mode)

# batch.py local fallback
replacements = load_replacements_from_file()
pipeline = build_pipeline(replacements)

result = engine.transcribe(filepath)
md = pipeline.run(result["text"])

Same pipeline, replacements from ~/.resona/replacements.json

replacements.json Format

~/.resona/replacements.json
[

  { "name": "mfg",       "replacement": "Mit freundlichen Grüßen" }
]

Same format as what ws-api currently stores in SQLite and sends to the engine. Direct compatibility.
